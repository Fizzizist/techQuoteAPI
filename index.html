Tech Quote API by Peter Vlasveld

As I was completing the random quote machine project at freeCodeCamp, I decided that I wanted to use tech quotes that I could relate to so that I could also add my quote machine to my portfolio page.
I searched for a simple and free tech quote API online but I couldn't find one, so I decided that I would just build my own. I did some research on Node.js and set up this API using my previous experience with MySQL.

I am sharing this API free for anyone to use in case you are also looking for a JSON API that randomly generates tech quotes. So enjoy:

Endpoint: http://groupstack.net/techQuoteAPI/api/
